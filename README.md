# Junk Removal GTA

A Next.js 14 lead generation site for junk removal services in the Greater Toronto Area.

## Features

- ✅ Next.js 14 with App Router
- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ Markdown content support (gray-matter, marked)
- ✅ Clean folder structure

## Project Structure

```
junk-removal-gta/
├── app/              # Next.js App Router
├── components/       # Reusable React components
├── content/
│   └── cities/      # City-specific content (markdown files)
├── package.json
└── ...
```

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Building for Production

```bash
npm run build
npm start
```
